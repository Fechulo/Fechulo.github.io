Command List

Below is a table of all available commands, including a brief description of the command and its syntax and arguments:

Command Syntax

Description

.VECP CHARTS [ICAO]

Opens a browser tab to the IAA charts for the specified airport.

.VECP METAR [ICAO]

Prints the latest METAR for the specified airport.

.VECP TAF [ICAO]

Prints the latest TAF for the specified airport.

.VECP TIMER [SECONDS]

Creates a custom countdown timer for the specified time (in seconds).

.VECP HELP

Prints information about useful commands to find out more about the plugin.

.VECP GUIDE

Opens the VÃ‰CP Controller Guide in the default PDF reader.

.VECP COMMANDS

Opens the VÃ‰CP Commands Reference in the default PDF reader.

.VECP SETTINGS

Opens the VÃ‰CP Settings Reference in the default PDF reader.

.VECP DEBUG

Toggles the debug setting on and off.

.VECP OCEAN [CALLSIGN]

If you think the plugin should have recognised it as such, please send the printed debug information as well as a screenshot of the aircraft, its flight plan and route to the plugin developer for debugging purposes.
