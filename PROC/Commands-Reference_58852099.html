Introduction

The VATéir Controller Plugin introduces a number of commands that can be used through the EuroScope chat. For compatibility with aliases and other plugins, all VÉCP commands follow this format:

.VECP [command] (argument)

the plugin ignores case so capitalization does not affect any commands; YoU cAN tYpE LIke ThIS if yOu lIke

Command List

Below is a table of all available commands, including a brief description of the command and its syntax and arguments:

Command Syntax

Description

.VECP CHARTS [ICAO]

Opens a browser tab to the IAA charts for the specified airport.

.VECP METAR [ICAO]

Prints the latest METAR for the specified airport.

.VECP TAF [ICAO]

Prints the latest TAF for the specified airport.

.VECP TIMER [SECONDS]

Creates a custom countdown timer for the specified time (in seconds).

.VECP HELP

Prints information about useful commands to find out more about the plugin.

.VECP GUIDE

Opens the VÉCP Controller Guide in the default PDF reader.

.VECP COMMANDS

Opens the VÉCP Commands Reference in the default PDF reader.

.VECP SETTINGS

Opens the VÉCP Settings Reference in the default PDF reader.

.VECP DEBUG

Toggles the debug setting on and off.

.VECP OCEAN [CALLSIGN]

Prints debug information regarding the specified aircraft’s oceanic data and forces the specified aircraft to be considered oceanic traffic. Can be used for aircraft with strange routes that the plugin did not recognise as oceanic traffic. If you think the plugin should have recognised it as such, please send the printed debug information as well as a screenshot of the aircraft, its flight plan and route to the plugin developer for debugging purposes.

.VECP OWNER [CALLSIGN]

Prints debug information regarding the specified aircraft’s plugin ownership. If you think the printed information is incorrect, please send this information as well as a screenshot of the aircraft, its flight plan and route to the plugin developer for debugging purposes.
